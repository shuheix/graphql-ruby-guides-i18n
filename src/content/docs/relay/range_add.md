Relay specifies `RANGE_ADD` operations for adding items to connections. GraphQL-Ruby ships with [`GraphQL::Relay::RangeAdd`](https://graphql-ruby.org/api-doc/GraphQL::Relay::RangeAdd) to help implement this. Check the API docs for a usage example.
