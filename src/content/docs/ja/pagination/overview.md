GraphQL-Ruby は Relay の "connection" スタイルのページネーションを複数の実装とともに提供します。まず [Connection の概念](/pagination/connection_concepts) で connection に慣れ、[Connections の使い方](/pagination/using_connections) で利用方法を確認してください。カスタムの connection 実装や type 定義もサポートしており、詳細は [Custom Connections](/pagination/custom_connections) ガイドでご覧ください。

さらなる参考として、GraphQL の公式にある優れたページネーションのドキュメントもあります: [優れたページネーションのドキュメント](https://graphql.org/learn/pagination/).
