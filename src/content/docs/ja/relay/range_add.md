Relay は connections に項目を追加するための `RANGE_ADD` 操作を規定しています。GraphQL-Ruby にはこれを実装するための [`GraphQL::Relay::RangeAdd`](https://graphql-ruby.org/api-doc/GraphQL::Relay::RangeAdd) が含まれています。使用例は API ドキュメントを参照してください。
